HSL color

HSL colors can be defined through three values. The first represent the hue of the color, and occupies a range from 0 to 360, indicating the position of the color on a color wheel. The second value is saturation, and it indicates how saturated with color the value is. Saturation is represented as 0% to 100%, with 100% being fully saturated and 0% being grayscale. The final value represents lightness, and represents the darkness value of the color. 0% indicates black, while 100% indicates a bright color.
